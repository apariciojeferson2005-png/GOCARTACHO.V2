Política de Privacidad

Última actualización: Mayo 2026

Compromiso Principal

La privacidad y seguridad de tus datos son nuestra prioridad absoluta. Los datos de ubicación utilizados para generar los mapas de calor se procesan de forma completamente anónima y agregada, haciendo imposible la identificación de cualquier individuo.

1. Datos que Recopilamos

  • Datos de ubicación anónimos: Coordenadas geográficas generalizadas que se asocian a un hash de dispositivo irreversible. No se vinculan a tu identidad personal.
  • Datos de registro: Email, nombre y contraseña (encriptada con BCrypt) cuando creas una cuenta voluntariamente.
  • Datos de interacción: Reseñas y calificaciones que publicas voluntariamente sobre negocios.

2. Cómo Usamos los Datos

  • Mapas de calor: Los datos de ubicación se agregan estadísticamente para mostrar densidades de personas en zonas geográficas. Nunca se muestran posiciones individuales.
  • Planes populares: Se utilizan datos históricos agregados para sugerir itinerarios optimizados.
  • Mejora del servicio: Los datos agregados nos ayudan a mejorar la experiencia de usuario.

3. Protección de Datos

  • Las contraseñas se almacenan con encriptación BCrypt (estándar de la industria).
  • Los datos de ubicación se procesan con hashes irreversibles que impiden la re-identificación.
  • Las sesiones de usuario se gestionan de forma segura con timeout automático.
  • Las cookies de sesión están configuradas como HTTP-only para prevenir ataques XSS.

4. Compartición de Datos

No vendemos, alquilamos ni compartimos tus datos personales con terceros. Los datos de afluencia son puramente estadísticos y se utilizan exclusivamente dentro de la plataforma GO CARTACHO.

5. Tus Derechos

  • Acceso: Puedes solicitar una copia de tus datos personales.
  • Rectificación: Puedes actualizar tu información en cualquier momento.
  • Eliminación: Puedes solicitar la eliminación de tu cuenta y datos asociados.
  • Oposición: Puedes dejar de contribuir datos de ubicación desactivando la geolocalización de tu dispositivo.

6. Datos de Negocios

Los negocios registrados en la plataforma comparten voluntariamente su información de contacto, horarios y promociones. Esta información es pública y visible para todos los usuarios de la aplicación.

7. Contacto

Para consultas relacionadas con la privacidad, puedes contactarnos a: privacidad@gocartacho.com
